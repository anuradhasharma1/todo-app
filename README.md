Todo App (React + Vite)

A clean and minimal **Todo App** built using **React** and **Vite**.  
Designed to manage daily tasks efficiently with a smooth and fast UI.

##  Features
-  Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  Tasks persist using Local Storage
-  Super fast dev experience with Vite
-  Responsive design

##  Tech Stack
- **React** – UI library
- **Vite** – Build tool
- **JavaScript (ES6+)**
- **CSS / Tailwind CSS**
